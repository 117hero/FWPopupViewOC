## 支持pod导入：

```cocoaPods
pod 'FWPopupViewOC'
注意：如出现 [!] Unable to find a specification for 'FWPopupViewOC' 错误，可执行 pod repo update 命令。
```



## 简单使用：

### OC：<br>
```oc
只需要FWPopupBaseView：

@interface FWCustomView : FWPopupBaseView
@end

```
